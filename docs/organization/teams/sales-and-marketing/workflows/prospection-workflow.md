# Flujo de Prospección de Oportunidades B2B de 23people

## Propósito

El propósito de este flujo es establecer un proceso estructurado para la
prospección de oportunidades de negocio B2B, asegurando una gestión eficiente
y efectiva desde la identificación inicial hasta la calificación de las
oportunidades para su posterior seguimiento por ventas.
